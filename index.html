Installation

Install Aroma.js using npm:

npm install aroma.js

Routing

Create routes easily:

const app = new Aroma();

app.get('/hello', (req, res) => {
    res.send({ message: 'Hello, World!' });
});

app.listen(3000, () => console.log('Server running on port 3000'));

Router

Create modular routers for better organization:

import { Router } from 'aroma.js';

const router = new Router();

router.get('/', (req, res) => {
    res.send({ message: 'Hello from Router!' });
});

router.get('/:id', (req, res) => {
    res.send({ message: `User ID: ${req.params.id}` });
});

export default router;

Mount the router in your application:

import Aroma from 'aroma.js';
import userRouter from './routes/user';

const app = new Aroma();
app.use('/users', userRouter);

app.listen(3000, () => console.log('Server running on port 3000'));

Middleware

Use middleware to enhance functionality:

app.use((req, res, next) => {
    console.log(`${req.method} request to ${req.url}`);
    next();
});

Session Management

Manage sessions seamlessly:

app.useSessions();

app.get('/set-session', (req, res) => {
    req.session.username = 'JohnDoe';
    res.send({ message: 'Session set' });
});

app.get('/get-session', (req, res) => {
    res.send({ username: req.session.username });
});

Template Engine

Render dynamic templates:

app.enableTemplateEngine();

app.get('/home', (req, res) => {
    app.render(res, 'index', { title: 'Welcome', user: 'John' });
});

Get Started

Install Aroma.js and start building your application in minutes.

npm install aroma.js
